    Panduan Teknis & Pengelolaan Steering Committee

    1. Menambahkan Program Kerja

    Secara Satuan: Klik tombol "Tambah Proker Baru" di section Program Kerja. Isi Nama (Wajib), Tanggal, Divisi, dan PIC. Anda juga bisa langsung menambahkan jadwal rapat di form tersebut.

    Secara Massal (Bulk): Gunakan tombol "Tambah Massal". Fitur ini sangat berguna untuk input banyak proker sekaligus (misal: saat awal periode kabinet). Klik "Simpan Semua" setelah selesai mengisi semua baris.

    2. Mengedit & Administrasi

    Ceklis Administrasi: Anda bisa langsung mencentang kolom Proposal, RAK, RAB, dan LPJ di Tabel Utama. Perubahan ini akan memicu "Batch Save" (tombol biru di bawah) agar hemat kuota kiriman ke server.

    Edit Keseluruhan: Klik tombol "Edit" di daftar proker. Di sini Anda bisa mengubah Nama, Divisi, hingga menambah/menghapus "Jadwal Rapat" yang spesifik untuk proker tersebut.

    3. Struktur & Detail Database (Spreadsheet)

    Sistem ini menggunakan Google Sheets sebagai database. Berikut adalah rincian kolom dan cara pengisiannya agar tidak terjadi error:

    Sheet: Proker (Data Utama)
    • ID: (Otomatis) ID unik bersifat permanen. Jangan diubah kecuali terpaksa.
    • nama: Judul kegiatan. Contoh: Latihan Kepemimpinan Mahasiswa.
    • divisiId: ID Divisi (harus ada di MasterDivisi). Contoh: D01.
    • picId: ID PIC (harus ada di MasterPIC). Contoh: P01.
    • tanggal: Format YYYY-MM-DD. Contoh: 2024-03-24.
    • proposal, rak, rab, lpj: Status centang. Isi TRUE (tercentang) atau FALSE (kosong).
    • statusSelesai: Akan otomatis TRUE jika 4 kolom admin di atas semuanya TRUE.
    Sheet: rapat_proker (Jadwal Rapat)
    • idProker: Harus sama persis dengan ID di sheet Proker agar terhubung.
    • jenisRapat: Kategori rapat. Contoh: Rapat Perdana / Rapat Pleno.
    • tanggalRap: Tanggal rapat. Format DD/MM/YYYY. Contoh: 20/03/2024.
    • pic / picEmail: Nama dan email yang akan muncul di notifikasi.
    Sheet: MasterPIC, MasterDivisi, & MasterRapat
    • MasterPIC: Tambah Nama Pengurus agar muncul di pilihan PIC Web. Contoh: P15 | Andi | D02 | andi@mail.com.
    • MasterDivisi: Tambah nama departemen baru. Contoh: D07 | Pengembangan Bisnis.
    • MasterRapat: Tambah kategori rapat baru agar muncul di dropdown. Contoh: Rapat Koordinasi Wilayah.
    Sheet: MasterSC (Data Login)

    Sheet ini sangat rahasia. Digunakan untuk autentikasi login pengurus SC.

    • username: Digunakan untuk login.
    • password: Kata sandi login.
    • nama: Nama pengurus yang akan muncul di sistem (Audit Trail).
    ⚖️ Aturan Pengeditan Langsung (Spreadsheet vs Web):
    • Sangat Aman: Mengedit isi sel data (seperti typo nama, koreksi tanggal, atau ganti status centang) langsung di spreadsheet. Pastikan format tetap konsisten.
    • Gunakan Web: Untuk menambah Proker atau Rapat baru. Web menjamin ID tergenerasi otomatis secara unik agar data tidak tertukar.
    • Dilarang Keras: Mengubah Nama Judul Kolom (Baris 1), Menghapus/Menyisipkan Kolom tengah, atau Mengubah ID yang sudah ada karena akan merusak sinkronisasi antar sheet.

    4. Peringatan Penting

    Jangan Hapus Baris Secara Sembarangan: Menghapus baris di spreadsheet tanpa melalui Web dapat menyebabkan sinkronisasi ID bermasalah.

    Aktivitas Tercatat: Setiap perubahan (Create, Update, Delete) yang dilakukan di Mode SC akan mencatat Username dan Waktu di sistem sebagai bukti pertanggungjawaban.

    Selalu Backup: Sebelum melakukan perubahan besar di spreadsheet langsung, pastikan Anda mencadangkan file atau menduplikasi sheet tersebut.

    Panduan Penggunaan HIMAGRO HUB

    Hak Akses & Navigasi

    Status: Pengguna Umum (Staff) - Hak Akses "Lihat Saja".

    • Section Proker: Pantau progres administrasi (Proposal hingga LPJ) setiap kegiatan. Klik nama proker untuk rincian jadwal rapat.
    • Section Kalender Pengurus: Pantau agenda organisasi dan jadwal rapat rutin pengurus secara visual.
    • Section Konten: Pantau jadwal publikasi media sosial agar tidak terjadi bentrok informasi.

    Fitur & Fungsi Tombol

    Refresh Data: Mengambil data terbaru dari server jika pengurus SC baru saja melakukan update.
    Reset Aplikasi: Gunakan jika aplikasi terasa error atau data tidak mau muncul (Clear Cache).
    Mode SC (Admin): Tombol login khusus pengurus untuk manajemen data kegiatan.
    Instal Aplikasi: Android/PC: Klik tombol di atas (jika muncul).
    iPhone (iOS): Klik tombol Share/Bagikan di Safari, lalu pilih "Add to Home Screen".
